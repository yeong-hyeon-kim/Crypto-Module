## 📕 프로젝트 개요(Introduce Project)

### Crypto Module

* .NET 암호화 클래스를 이용한 텍스트 암호화 모듈입니다.

## 🏷️ 기능(Function)

2. [HASH(해시 암호화)](#HASH(해시-암호화))
3. [RSA(공개키 암호화)](#RSA(공개키-암호화))

### 세부 기능(Function Detail)

#### AES(대칭키 암호화)

   1. AES 256 암호화
   2. AES 256 복호화

#### HASH(해시 암호화)

   1. SHA 128 암호화
   2. SHA 256 암호화
   3. SHA 384 암호화
   4. SHA 512 암호화

#### RSA(공개키 암호화)

   1. RSA 암호화
   2. RSA 복호화
